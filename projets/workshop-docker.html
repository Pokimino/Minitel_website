Workshop Docker

Formation pratique sur la conteneurisation

Le Workshop Docker organisé par MINITEL est une formation pratique qui permet aux étudiants de découvrir et maîtriser la conteneurisation avec Docker. Cette technologie révolutionne le déploiement d'applications et est devenue incontournable dans le développement moderne.

Contenu du workshop :

  • Introduction à la conteneurisation
  • Installation et configuration de Docker
  • Création et gestion de conteneurs
  • Développement d'applications conteneurisées
  • Orchestration avec Docker Compose
  • Bonnes pratiques et sécurité
